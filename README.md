# MaskLens
An AI-powered scam and misinformation detector for Africans. Helps Africans detect scams, misinformation, and AI-manipulated content.

## Problem
Digital fraud and misinformation are increasing across Africa.

## Solution
Users upload suspicious messages, job offers, screenshots, or media, and the AI analyzes them for scam indicators and manipulation patterns.
